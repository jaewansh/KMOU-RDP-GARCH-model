# KMOU-RDP-GARCH-model

Python code for reproducing the main and supplementary empirical results in the manuscript:

**Persistence Misspecification and Post-Crisis Volatility Overestimation: A Regime-Dependent GARCH Approach**

## Overview

This repository contains Python scripts used to reproduce the main and supplementary analyses for the proposed **Regime-Dependent Persistence GARCH (RDP-GARCH)** framework.

The study investigates why standard single-regime GARCH models tend to overestimate volatility after major financial shocks and shows that this problem is primarily driven by **persistence misspecification**. To address this issue, the RDP-GARCH framework allows persistence to vary across regimes and provides a more structurally coherent description of post-crisis volatility adjustment.

The empirical analysis is based on daily stock index data for:

- **KOSPI**
- **S&P 500**

The supplementary analysis extends the empirical evidence to additional crisis episodes, including the **2008 Global Financial Crisis** and the **COVID-19 shock**.

## Scripts

### `code/empirical_IMP.py`
Main empirical script for the manuscript.

This script is intended to reproduce the **main-text empirical figures and results**.  
Figures generated by this script are saved to:

```text
data/figure/
```

### `code/empirical_supp.py`
Supplementary empirical script.

This script is intended to reproduce the **supplementary figures and results**.  
Figures generated by this script are saved to:

```text
data/figure/supplement/
```

## Path Convention

All executable Python scripts are assumed to be located directly under the `code/` directory and use repository-relative paths.

```python
import os

# Repository root
BASE_DIR = os.path.dirname(os.path.dirname(os.path.abspath(__file__)))

# Main directories
CODE_DIR = os.path.join(BASE_DIR, "code")
DATA_DIR = os.path.join(BASE_DIR, "data")
FIG_DIR = os.path.join(DATA_DIR, "figure", "supplement")
```

In practice:

- main manuscript figures are saved under `data/figure/`
- supplementary figures are saved under `data/figure/supplement/`

This repository structure avoids machine-specific absolute paths and improves portability across environments.

## Data

Raw data are not bundled with this repository by default.

The empirical analysis relies on daily historical index data for **KOSPI** and **S&P 500**, as described in the manuscript. Users should prepare the required input data locally and place them under the `data/` directory before running the scripts.

If necessary, users may organize raw or intermediate files within `data/` while preserving the output directory structure for figures.

## Reproducing the Results

A typical workflow is:

1. Clone this repository.
2. Prepare a Python environment and install the required packages.
3. Place the input data files under the `data/` directory.
4. Run the main empirical script:
   ```bash
   python code/empirical_IMP.py
   ```
5. Run the supplementary empirical script:
   ```bash
   python code/empirical_supp.py
   ```

## Outputs

This repository is designed to reproduce figures and empirical evidence related to:

- post-crisis volatility adjustment,
- regime-dependent persistence,
- comparisons across markets and crisis episodes,
- time-varying probabilities of the high-volatility regime.

The main script produces the figures for the main manuscript, while the supplementary script produces the figures for the supplementary material.
